**Query Parameters:**

- `campaign_id` (optional): Filter by specific campaign

**Response Schema:** `ProcessedReplyStats`
```json
{
  "total": 0,
  "by_category": {"interested": 5, "not_interested": 2, ...},
  "by_status": {"pending": 3, "approved": 4, "dismissed": 1},
  "today": 0,
  "this_week": 0,
  "sent_to_slack": 0,
  "pending": 0,
  "approved": 0,
  "dismissed": 0
}
```

## What the endpoint provides:
1. **Total count** - All processed replies
2. **by_category** - Breakdown by AI classification (interested, not_interested, etc.)
3. **by_status** - Breakdown by approval status (pending, approved, dismissed)
4. **today** - Replies processed today
5. **this_week** - Replies processed this week
6. **sent_to_slack** - Count of replies sent to Slack
7. **pending/approved/dismissed** - Quick access counts
