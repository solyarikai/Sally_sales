# Product Context for Replies

## OnSocial — What We Sell

OnSocial is a B2B data infrastructure platform for influencer marketing.

### Core Value Props by Segment

**For IM Agencies (IMAGENCY):**
- White-label creator data — your brand, your pricing, your client relationship
- API or dashboard with agency's logo
- Clients see agency as data source, not a middleman
- Pain: agencies use HypeAuditor/Modash, clients see competitor brand in reports
- Hook: 67% of brands considering moving IM in-house — agencies that own data survive

**For Influencer Platforms (INFPLAT):**
- 300M+ (or 450M+) profiles across Instagram, TikTok, YouTube
- Full demographics, fraud detection, audience overlap, brand affinities
- One API call replaces 6+ months of in-house data engineering
- Pain: building creator data layer in-house takes 6+ months eng time
- Hook: one integration, all markets covered

**For Multi-Market Platforms (MENA/APAC):**
- Cross-market coverage through single API
- Geo-breakdown by country and city
- Normalized metrics across platforms and regions
- Pain: fragmented data across TikTok (SEA), Instagram (MENA), YouTube (AU)

### Key Proof Points
- 300M+ / 450M+ creator profiles
- Audience demographics down to city level
- Fraud detection / credibility scoring
- Cross-platform (IG, TikTok, YouTube)
- Most integrations go live in under 2 weeks
- Clients: Lefty (freed 2 eng roles), and others

### Free Offers (for exit frames)
- Free data audit: drop 5 creator handles, get full analytics breakdown
- Run any creator handle through the system live
- Technical demo with real API response payloads
