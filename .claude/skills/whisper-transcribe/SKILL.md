# Whisper Transcribe

Локальная бесплатная транскрипция: файлы, YouTube-видео, плейлисты, каналы.

**Whisper**: `/Users/user/Library/Python/3.14/bin/whisper`
**yt-dlp**: для YouTube (если нет — `brew install yt-dlp`)

---

## Шаг 1 — Определи режим и параметры

Из контекста определи (не спрашивай, если очевидно):

**Режим:**
- **Файл** — путь к локальному аудио/видео
- **YouTube видео** — одиночная ссылка (содержит `watch?v=` или `youtu.be/` или `/shorts/`)
- **YouTube плейлист/канал** — содержит `playlist?list=`, `/@`, `/channel/`, `/c/` + `/videos`

**Параметры:**

| Параметр | По умолчанию |
|----------|-------------|
| Язык | Авто-определение (не указывай `--language`) |
| Модель | `turbo` |
| Формат | `txt` |
| Куда сохранить | Рядом с исходным файлом / текущая директория |

---

## Шаг 2 — Получи аудио

### Локальный файл
Ничего делать не нужно, переходи к Шагу 3.

### YouTube видео
Сначала проверь и установи `yt-dlp` если нужно:
```bash
which yt-dlp || brew install yt-dlp
```
Затем скачай аудио:
```bash
yt-dlp -f "bestaudio" -x --audio-format mp3 -o "/tmp/%(id)s.%(ext)s" "<URL>"
```
Результат: `/tmp/<video_id>.mp3`

### YouTube плейлист / канал
Проверь `yt-dlp` как выше, затем получи список видео:
```bash
yt-dlp --flat-playlist --print "%(url)s\t%(title)s" "<URL>"
```
Затем для каждого видео скачай аудио как выше. Сообщай прогресс: `Видео 1/N: <название>...`

---

## Шаг 3 — Транскрибируй через Whisper

```bash
/Users/user/Library/Python/3.14/bin/whisper "<путь>" --model turbo --output_format txt --output_dir "<папка>"
```

**С указанием языка:**
```bash
/Users/user/Library/Python/3.14/bin/whisper "<путь>" --model turbo --language ru --output_format txt --output_dir "<папка>"
```

**Перевод на английский (любой язык -> English):**
```bash
/Users/user/Library/Python/3.14/bin/whisper "<путь>" --model turbo --task translate --output_format txt --output_dir "<папка>"
```

**Субтитры:**
```bash
/Users/user/Library/Python/3.14/bin/whisper "<путь>" --model turbo --output_format srt --output_dir "<папка>"
```

После транскрипции YouTube — удали временный mp3:
```bash
rm /tmp/<video_id>.mp3
```

---

## Шаг 4 — Покажи результат

1. Прочитай полученный файл
2. Покажи транскрипт пользователю
3. Для YouTube — сохрани как `<slug>_transcript.md` в текущей директории
4. Если уместно — предложи сохранить в `sofia/transcripts/`

Для плейлиста — итоговый отчёт:
```
Транскрибировано: N видео

1. <название> → <путь к файлу>
2. <название> → <путь к файлу>
```

---

## Дедупликация (для плейлистов)

Перед транскрипцией проверь:
- Если `<slug>_transcript.md` уже существует — пропусти, сообщи: `Уже есть: <файл>, пропускаю.`
- Веди список обработанных video_id в рамках одного запуска

## Выбор модели

| Модель | Скорость | Качество | Когда |
|--------|----------|----------|-------|
| `tiny` | Мгновенно | Низкое | Черновик |
| `small` | Быстро | Хорошее | Короткие записи |
| **`turbo`** | **Быстро** | **Высокое** | **По умолчанию** |
| `large` | Медленно | Лучшее | Максимальное качество |

Если файл > 30 мин — предупреди, что займёт несколько минут.

## Форматы

`.mp3` `.mp4` `.wav` `.m4a` `.ogg` `.flac` `.webm` `.mkv` `.avi`
