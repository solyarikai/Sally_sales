# OnSocial — Project Overview

## What We Sell

OnSocial — B2B SaaS платформа creator/influencer analytics. Два основных продукта:
- **White-label API** для SaaS-платформ (встраивают данные OnSocial в свой продукт)
- **White-label Analytics Dashboard** для агентств (подают клиентам под своим брендом)

Ключевые характеристики: 450M+ профилей, IG/TikTok/YouTube, обновление каждые 24-48ч, покрытие 50+ стран на уровне города.

## Positioning by Segment

**Для платформ (INFPLAT):** "Зачем строить data pipeline самим? 2+ инженера освобождаются, coverage с day one вместо 4-6 месяцев разработки."

**Для агентств (IMAGENCY):** "White-label данные под вашим брендом. Клиенты не гуглят vendor price. 40% выше retention с branded analytics."

**Для affiliate (AFFPERF):** Performance-данные по creator marketing, интеграция с attribution стеками.

## Geographic Priority

По результатам кампаний:
1. **MENA + APAC** — 10x конверсия vs US/EU (8.7% meeting rate)
2. **India** — 100% warm rate на replies, white-label позиционирование конвертит лучше всего
3. **UK, Germany, France** — основные рынки Европы (45% от IMAGENCY dataset)
4. **US/EU** — большой TAM, но конверсия ниже

**Germany:** высокий юридический риск cold email (UWG requires prior consent) — использовать LinkedIn.
