# Сегменты для масштабирования — Международный рынок

Анализ на основе 14 квалифицированных лидов из leads_research.docx

---

## Сегмент 1: Недвижимость за рубежом (7 из 14 — основной массив)

**Компании из квалов:** Dukley Gardens, Housebook, Primal Alliance, Bricksfield, My Portugal RE, Excel Properties, UDI Group

**Суть:** Агентства, брокеры и девелоперы, чьи клиенты покупают недвижимость за рубежом. Основные ГЕО — ОАЭ (Дубай), Черногория, Португалия, Чехия/CEE.

**Почему им нужен Deliryo:**
- Клиенты из РФ/СНГ хотят оплатить недвижимость, но не могут сделать прямой перевод
- Сделки подвисают из-за проблем с платежами — агентство теряет комиссию
- Deliryo решает: RUB -> USDT за 3 часа + документы для банка

**Как масштабировать:**
- ОАЭ/Дубай — самый жирный подсегмент (4 компании уже в списке). Десятки русскоязычных агентств, RERA-реестр брокеров — готовая база
- Черногория / Португалия / Испания / Греция — рынки Golden Visa, масса русских покупателей
- Кипр / Турция / Таиланд / Бали — популярные направления для релокантов из РФ

**Оффер:** "Ваш клиент оплачивает из российского банка — получаете закрытую сделку + 0.2% сверху"

## Сегмент 2: Инвестиционная миграция (2 из 14)

**Компании из квалов:** Orience, Jurisprudential

**Суть:** Помогают получить ВНЖ/гражданство через инвестиции (Golden Visa, CBI-программы). У клиентов есть деньги в РФ, которые нужно перевести для квалификации.

**Почему им нужен Deliryo:**
- Клиент хочет Golden Visa в Испании — нужно инвестировать 500K EUR — деньги в российском банке — стандартный перевод невозможен
- Миграционная компания теряет клиента, если не может решить вопрос с оплатой

**Как масштабировать:**
- Базы членов IMI (Investment Migration Insider), IIUSA
- CBI-агентства на Карибах, в Европе, Турции
- Каталоги Golden Visa агентов в Испании, Португалии, Греции
- LinkedIn: "immigration consultant", "citizenship by investment"

**Оффер:** "Помогаем вашим клиентам из РФ выполнить инвестиционный порог для ВНЖ — перевод за 3 часа с документами"

## Сегмент 3: Wealth Management / Страхование / Family Offices (3 из 14)

**Компании из квалов:** 1291 Group, Swiss Sure Group, Yokey

**Суть:** Управляют состоянием HNWI/UHNWI — страхование жизни (PPLI), трасты, estate planning, инвестиции.

**Почему им нужен Deliryo:**
- Клиенты-россияне хотят пополнить полис PPLI, инвестиционный портфель или траст — не могут перевести из РФ
- Средний чек значительно выше ($300K-$5M+)
- B2B модель: они работают с private банками и другими посредниками — один партнёр = поток сделок

**Как масштабировать:**
- Швейцарские и сингапурские wealth-менеджеры (LinkedIn, STEP.org, APCWM)
- Family offices каталоги (Simple, Family Office Exchange)
- PPLI провайдеры (Lombard International, Investors Trust)
- Private банки с русскоговорящими десками (Julius Baer, UBS, Pictet)

**Оффер:** "Ваши российские клиенты могут пополнить полис/траст через USDT с полным пакетом документов. Партнёрская комиссия до 0.5%"

## Сегмент 4: Юридический и корпоративный консалтинг (3 из 14)

**Компании из квалов:** GLG Consult, Enalian, Jurisprudential (пересекается с Сегментом 2)

**Суть:** Помогают структурировать бизнес за рубежом — открытие компаний, банковские счета, налоговое планирование, крипто-лицензирование.

**Почему им нужен Deliryo:**
- Клиенты открыли компанию за рубежом — нужно её капитализировать — деньги в РФ
- Решение проблемы "деньги в РФ, бизнес за рубежом"
- GLG Consult прямо упоминает крипто-легализацию и платёжного агента

**Как масштабировать:**
- Русскоязычные юр. фирмы: Кипр, ОАЭ, Эстония, Грузия, Сербия
- Каталоги offshore-провайдеров (Offshore Company, Flag Theory)
- Бали/ЮВА — волна русских предпринимателей
- LinkedIn: "corporate structuring", "tax planning", "offshore company formation"

**Оффер:** "Клиенты регистрируют компанию — нужно перевести капитал из РФ — мы делаем это за 3 часа с документами для банка и налоговой"

## Сегмент 5: PropTech / Digital платформы (1 из 14, но масштабируемый)

**Компания из квалов:** Housebook

**Суть:** Платформы, которые агрегируют брокеров/застройщиков. Одна интеграция = доступ к сотням агентов.

**Как масштабировать:**
- PropTech платформы ОАЭ и Юго-Восточной Азии
- Платформы типа Tranio, Prian, Realting — агрегаторы зарубежной недвижимости для русскоязычных
- Crypto-friendly платформы покупки недвижимости (Propy, RealT)

**Оффер:** "Встраиваем платёжное решение RUB -> USDT прямо в вашу платформу — ваши брокеры закрывают больше сделок"

## Главный вывод

Квалы сильно перекошены в недвижимость (50%) — это основной product-market fit. Для масштабирования: начать с насыщения ГЕО Дубай (больше всего русскоязычных агентств), потом Кипр, Черногория, Турция, и параллельно тестировать wealth/миграционный сегмент — там чеки кратно выше.
