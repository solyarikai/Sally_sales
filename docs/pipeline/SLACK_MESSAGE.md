*TAM Gathering Pipeline*

Теперь есть общий пайплайн для поиска целевых компаний. Работает через Claude Code, не надо писать скрипты.

*Как начать:*
```
git pull && claude --dangerously-skip-permissions
```

Скажите Claude что искать. *Обязательно назовите проект:*
> Найди digital-агентства в ОАЭ до 50 человек для *easystaff global*
> Импортируй таблицу для *inxy*: <ссылка на Google Sheet>
> Поищи в Clay fintech в Сингапуре для *tfp*

Claude сам запустит пайплайн и остановится на 3 чекпоинтах:
1. Подтвердить проект и кампании
2. Просмотреть список целевых компаний
3. Одобрить стоимость FindyMail

*Если Claude пишет скрипт вместо пайплайна:*
> Используй gathering pipeline, не пиши скрипт. Посмотри CLAUDE.md.

*Если что-то зависло:*
> Проверь незавершенные запуски для [проект]

Все скрипты запускаются на сервере, не локально. Claude делает это автоматически.

Полная инструкция: `docs/pipeline/OPERATOR_GUIDE.md`
Техническая документация: `docs/pipeline/TAM_GATHERING_ARCHITECTURE.md`
