# Crona.ai — Продукт для сбора и обогащения B2B-данных

## Что такое Crona.ai

Crona.ai — это AI-платформа для web scraping и обогащения контактных данных, созданная командой Sally AI. Их слоган: "Instantly turn websites and directories into structured ICP lists with contacts." Используется 50+ sales-командами, подтверждена 20+ публичными кейсами.

Crona — прямой конкурент Clay, но позиционируется как более простой и дешёвый инструмент для тех же задач.

## Три основные функции

### 1. Scrape — Сбор данных с веб-сайтов
Превращает любой сайт, директорию или маркетплейс в чистые, структурированные списки лидов:
- Поддержка собственных прокси или встроенный proxy Crona
- Гибкие настройки под задачи продаж
- Данные собираются безопасно через аккаунт пользователя

**Источники данных:**
- Любые веб-сайты и директории
- Sales Navigator (через Sales Navigator Agent)
- Конференционные приложения (через Chrome extension Crona Scraper)
- Telegram-чаты (через Chrome extension для web.telegram.org/a)
- CSV-файлы (загрузка своих данных для обогащения)
- Скоро: Airbnb, Google Maps, Amazon, Instagram

### 2. Filter — AI-фильтрация и сегментация
- **AI-промпты:** описываешь словами, по каким критериям сегментировать → AI анализирует сайты компаний и распределяет по категориям
- **Python-код:** для продвинутых пользователей — кастомная фильтрация через код
- **Стандартные фильтры:** по индустрии, размеру, географии

**Пример из практики Sally:**
Загружаешь 10,000 компаний → описываешь: "раздели на iGaming operators, iGaming platforms и payment orchestration, дополнительно отметь тех, кто уже работает с криптой" → AI анализирует сайты каждой компании → получаешь 50+ готовых сегментов с персональными офферами.

### 3. Enrich — Обогащение контактами
- Верифицированные рабочие email-адреса
- Прямые телефонные номера
- LinkedIn-профили (из имени, email, Twitter, сайта — из любых сырых данных)
- Данные о компании (размер, индустрия, технологии)
- **Waterfall enrichment:** используют 15+ провайдеров данных последовательно

## Уникальные функции Crona

### LinkedIn Activity Score
Алгоритм скорит контакты по их активности в LinkedIn от 1 до 10:
- Анализирует историю публикаций, количество коннекшенов и фолловеров
- Позволяет писать только активным пользователям
- На проектах Sally дал прирост конверсий в 2–3x при отсечении score ниже 5

### LinkedIn Finder
Из любых сырых данных находит LinkedIn-профиль:
- Имя + компания/сайт
- Рабочая почта
- Сайт + должность
- Twitter-аккаунт
- Или всё вперемешку (CSV с разнородными данными)

### Chrome Extension — Crona Scraper
Бесплатное расширение для Chrome:
- **Конференции:** заходите в веб-версию приложения конференции → запускаете парсер → собирает всех участников → обогащаете в Crona
- **Telegram:** заходите в web.telegram.org/a → открываете чат → парсер собирает данные участников → обогащаете LinkedIn'ами и рабочими почтами

### Call AI Enricher
AI-анализ сайтов компаний для сегментации — автоматически определяет:
- Бизнес-модель компании
- Технологический стек
- Релевантность вашему ICP

## Crona vs Clay — ключевые отличия

| Параметр | Crona.ai | Clay |
|----------|----------|------|
| **Позиционирование** | Web scraping + enrichment для профи аутрича | Data orchestration платформа |
| **Сложность** | Проще, заточен под SDR/sales teams | Сложный, требует RevOps/инженерных навыков |
| **Цена** | Дешевле (Crona позиционируется как альтернатива) | $134+/мес (annual), CRM-интеграции от $720/мес |
| **Scraping** | Встроенный (любые сайты, конференции, Telegram) | Нет нативного скрапинга |
| **Data providers** | 15+ провайдеров для waterfall enrichment | 75+ провайдеров |
| **AI-фильтрация** | AI-промпты + Python | Claygent + формулы |
| **Кому подходит** | Аутрич-агентства, SDR-команды | RevOps, GTM-инженеры |

**Ключевое отличие:** Crona покрывает полный цикл "от сайта до контакта" — scraping + фильтрация + enrichment в одном месте. Clay больше про оркестрацию данных из внешних источников.
