### Steps

0. All of the steps bellow can be done using crona api - contact Amir for access to the api. 
1. Export company websites from Apollo using suitable keywords. Use Apollo account provided, the companies export is credits-free, so export only companies not people. (Use [crona scraper](https://chromewebstore.google.com/detail/crona-scraper/eghllimhkgfalogohpndndjhdpikachc?authuser=0&hl=en&pli=1) to scrape apollo) 
2. Segment company websites through Crona using [**the segmentation enricher**](https://www.notion.so/ArchiStruct-Starter-Kit-2b5d8691cb4681158481d4f5cc3e663d?pvs=21). Sign up for Crona account and use your email. 
3. Remove companies categorized as OTHER.
4. Generate message personalized by a business-model in Crona, [**example**](https://www.notion.so/ArchiStruct-Starter-Kit-2b5d8691cb4681158481d4f5cc3e663d?pvs=21)
5. Generate subjects in Crona
6. Export contacts from Apollo for the relevant segments (excluding OTHER). (Use [crona scraper](https://chromewebstore.google.com/detail/crona-scraper/eghllimhkgfalogohpndndjhdpikachc?authuser=0&hl=en&pli=1) to scrape apollo) 
7. Merge contacts with the segment table so each contact has a "segment" column (the column name must be exactly "segment" for auto-analytics).
8. 100 contacts for each segment, 3 segments, 300 contacts total
9. Profit.
