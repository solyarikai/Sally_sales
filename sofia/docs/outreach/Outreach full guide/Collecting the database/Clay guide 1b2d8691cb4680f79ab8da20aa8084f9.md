# Clay guide

We generally use Clay through **trial accounts** because it’s more cost-effective. Instead of having one corporate account, we create multiple new accounts to extend trial periods and optimize credit usage.

### How to log into a new Clay account from the same device

To create and access a new Clay account on the same device without conflicts, you can:

1. **Сreate a separate Google account in a separate chrome profile**  and use it to log into Clay for the new profile.
2. **Use Dolphin Anty browser** and register the new account **via Google login** — this method works reliably.
    
    Install Dolphin Anty here: [https://dolphin-anty.com/ru/](https://dolphin-anty.com/ru/)
    

Both methods help avoid issues with session overlaps and make managing multiple accounts easier. You can get even more free credits by following the credit hack guide shared below. This helps stretch your usage without extra cost.
