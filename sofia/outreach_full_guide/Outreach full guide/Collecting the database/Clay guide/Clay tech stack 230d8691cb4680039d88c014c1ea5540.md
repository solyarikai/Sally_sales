# Clay tech stack

Guide: How to Find Companies by Tech Stack (via HG Insights in Clay → Apollo)

1. **Create a New Workbook in Clay**
    - Go to **Clay** and click **“New Workbook.”**
    - On the right-hand side, click **“All Sources.”**
    
    ![image.png](Clay%20tech%20stack/image.png)
    
2. **Select Data Source**
    - Search for and select **“Companies by Tech Stack with HG Insights.”**
3. **Apply Filters**
    - Set your filters based on:
        - **Product or technology name**
        - **Vendor / company using the tech**
    
    ![image.png](Clay%20tech%20stack/image%201.png)
    
4. **Export the List**
    - Once you’ve got the filtered list of companies, export the data.
5. **Upload to Apollo**
    - Import that company list into **Apollo**.
    - Use Apollo to **find and enrich contacts** from those companies
