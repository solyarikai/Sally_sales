## 🤖 Работа с энричерами

### 4️⃣ Фильтрация по результатам энричера (true/false)

**Задача:** Пропускать дальше только строки, где энричер вернул `true`

```jsx
{{result.callAi6?}}
```

> 📌 **Пример использования:** Когда энричер проверяет соответствие компании/роли вашему ICP. Где `callAi6` — название энричера.

**Реальный кейс:**

**Call AI Enricher (callAi6):**
```
Output TRUE if the input job title represents someone who would evaluate, purchase, 
or strongly influence the buying decision for a B2B data platform / lead generation solution, 
specifically targeting sales, business development, marketing, and growth decision-makers 
in B2B companies and agencies.
```

**Filter после энричера:**

ruby

`{{result.callAi6?}}`

> ✅ Результат: Только строки с true проходят дальше по workflow
> 

---

### 5️⃣ Исключение определенных сегментов

**Задача:** Отфильтровать нежелательные сегменты (например, "OTHER")

ruby

```jsx
!{{result.callAi4}}.include?("OTHER")
```

> ⚠️ **Важно:** Используется когда энричер выдает не просто true/false, а сегментирует компании. Все строки с меткой "OTHER" не пропускаются дальше.

**Реальный кейс:**

**Call AI Enricher (callAi4):**
```
You are an analyst classifying companies based on their marketing and lead generation services. 
Given a company website, output the most relevant segment name from the list below:

OUTREACH_AGENCY — B2B lead generation agencies that use COLD OUTREACH as their primary 
or one of their core services (email outreach, LinkedIn outreach, cold calling, multi-channel outreach).

OTHER — Everything else including: marketing agencies without cold outreach focus 
(SEO agencies, PPC agencies, branding agencies, content marketing agencies, social media agencies, 
website agencies, creative, digital marketing agencies), data platforms, software tools, consulting, 
app marketing businesses, and any company that doesn't do B2B cold outreach.
```

**Результат энричера:**

- Companies с OTHER сегментом

**Filter для исключения:**

```jsx
!{{result.callAi4}}.include?("OTHER")
```

Show Image

> ✅ Результат: Все компании с сегментом "OTHER" отфильтровываются, проходят только OUTREACH_AGENCY
> 

**Где:**

- `callAi4` — энричер, который сегментирует компании
- `!` — инверсия условия (NOT)
- `.include?()` — проверка вхождения подстроки

## 📞 Извлечение номеров телефонов с сайта

### 9️⃣ Все номера телефонов через запятую

**Задача:** Получить все телефонные номера с сайта для ручной проработки

`phone_regex = /
  (?:\+?(\d{1,3})[\s\-\.]?)?
  \(?(\d{1,4})\)?[\s\-\.]?
  (\d{1,4})[\s\-\.]?
  (\d{1,4})[\s\-\.]?
  (\d{1,9})?
/x

{{company.website_text}}.scan(phone_regex).map do |match|
  match.compact.join
end.uniq.join(", ")`

**Результат:**

`+14155551234, +442071234567, 8005551234`

> 📱 Use case: Полезен для колл-центров и команд SDR — видны все номера компании (главный офис, отделы продаж, поддержка)
> 

**Где:**

- `phone_regex` — сложное регулярное выражение, которое распознает различные форматы номеров
- Поддерживает: международные коды (+1, +44), скобки, дефисы, точки
- `.map.compact.join` — очищает и форматирует каждый номер
- `.uniq.join(", ")` — удаляет дубликаты и объединяет через запятую

---

### 🔟 Первый номер телефона

**Задача:** Получить только первый номер для мгновенного использования в аутриче

`phone_regex = /
  (?:\+?(\d{1,3})[\s\-\.]?)?
  \(?(\d{1,4})\)?[\s\-\.]?
  (\d{1,4})[\s\-\.]?
  (\d{1,4})[\s\-\.]?
  (\d{1,9})?
/x

match = {{company.website_text}}.scan(phone_regex).first
match ? match.compact.join : nil`

**Результат:**

`+14155551234`

> ⚡ Use case: Для автоматизированных cold calling кампаний — один номер в одной колонке, готов к импорту в диалер
> 

**Где:**

- `.first` — берет только первое совпадение
- `match ? ... : nil` — проверка на наличие результата (возвращает nil если номер не найден)
