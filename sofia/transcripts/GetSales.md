## 1. Sender Profiles — проверка аккаунтов

- Заходим в **Sender Profiles** — тут все подключенные аккаунты
- Проверяем статус: зеленая галочка = все ок. Бывает, что аккаунт разлогинивает — вместо "working" будет другой статус
- Если баги — есть чат в Телеграме напрямую с фаундером GetSales, они суперподробно саппортят

### Здоровье аккаунта и SSI

- У аккаунта есть шкала здоровья (придуманная GetSales)
- **Главная метрика — SSI** (Social Selling Index). На него ориентируемся при планировании capacity
- В клиентском Google Sheet (страница infrastructure) есть SSI-калькулятор: вписываешь SSI (например, 30) — он показывает недельный объем рассылки
- Два сценария: аккаунт использовался активно / не использовался

### Открытие аккаунта

Два способа:
1. Через **GoLogin**
2. Через **Cloud Browser** (удобнее — открывается во вкладке, можно вернуться в GetSales)

## 2. Автоматизации (Automation)

- Вкладка **Automation** — тут все флоу (воронки)
- Можно клонировать автоматизацию, чтобы не пересоздавать флоу заново
- Есть папки/теги для организации (например, "OnSocial", "PR firms", "Agency")

### Загрузка контактов

1. Кликаешь **Add Contacts** вверху автоматизации
2. Можно загружать лидов в уже имеющуюся историю
3. Отображается статус загрузки
4. Выбираешь аккаунты для рассылки

### Работа со списками

- Можно создать список, загрузить в него людей
- Потом в автоматизации просто выбрать этот список через фильтр слева
- При загрузке GetSales **enrichит контакты**: добавляет connections, информацию из LinkedIn
- Можно фильтровать: например, не отправлять людям с менее чем 500 connections

### Переменные в секвенциях

- Те же принципы, что в SmartLead — можно использовать переменные (имя, фамилия, название компании, ссылка на LinkedIn)
- **Важно:** название компании нужно предварительно нормализовать — прогнать через промт, который уберет лишнее

## 3. Dashboard — аналитика

- В **Dashboard** можно фильтровать по автоматизации и по периоду
- Сверху — данные в разрезе всех контактов за все время
- Ниже — разбивка по профилям
- Видна вся воронка: connections отправлено, replies и т.д.

---

## 4. Управление capacity

- Количество отправок регулируется на уровне **Sender Profile**
- Можно запускать сколько угодно кампаний — ограничение на уровне профиля, он просто не отошлет больше лимита
- Это безопасная история — если регулируешь на уровне профиля
