## Executive Summary — главное

1. **Заявленный ICP ≠ реальный ICP.** Сайт (02) продаёт «global SMB/startup platform с 360° compliance». Реальные конвертеры (01, 04, 06) — **service/agency/digital SMB 11–50 emp в Gulf/Cyprus/DACH, с русскоязычным founder-слоем**. 8 из 11 публичных кейсов имеют CIS-footprint [04:§10].
2. **Gulf = половина всего warm-реплая** (SA 28% + UAE 22% = 50% non-conf warm) [01:L11, GROWTH_STRATEGY]. При этом в EN-позиционировании Gulf не упомянут вообще [02:§6].
3. **Buyer — не CFO/CEO, а Finance Ops / Accountant / Payroll Manager / BizDev**. Named buyers: Julia Stradina (BizDev, Changer Club), Alia (Accountant, Proxeet) [04:§7]. В warm replies: Sumi Gulati (Veolia), Lawry Miller (ScaleMail) — middle-mgmt forward к founder.
4. **Best-performing sequence patterns (hard data, 06)**: DACH/CEO Ilya V. — **6.82% reply**, UAE-Pakistan v3 — **6.04%**, iGaming SIC Limassol — **3.5%**, IGB non-RUS — **3.97%**. Losers: generic geo-shard "Petr ES" — 0.3–0.9%.
5. **Sweet-spot размера**: 11–50 emp = 52% warm replies, 1–50 = 81% [01:L12]. 200+ почти не отвечают — не тратить credits.
6. **Топ-2 hook (доказанные)**: (а) "One invoice, one payment" (Achal/Frizzon close); (б) "Save 70% on commissions" (UAE outsourcing case) [01:§1.1-1.2]. Step 2 competitor-displacement — 31% warm replies.
7. **Критичное противоречие: UAE→India**. A1 Stanislav (Mar 18) говорит "не нужны — дешёвый банк-перевод". A1 GROWTH_STRATEGY (Mar 24) и A6 (UAE-PK v3) показывают 27/126 warm replies — топ-2 corridor. **Resolved**: fit есть НЕ на corridor-level, а на **service-agency business model** (UAE HQ, команда в Ливане/ЕС/РФ + контракторы в IN/PK).

## 9. Resolved Contradictions

### 9.1 UAE→India corridor: "не нужны" vs 27 warm replies ✅ resolved
- **Source A** (DUBAI_AGENCY_SEGMENT, Mar 18): "простой банк-перевод, не нужны" [01:§6.5]
- **Source B** (GROWTH_STRATEGY Mar 24, UAE-PK v3 Mar 17, UAE-India campaign 06): 27/126 warm replies, топ-2 corridor [01:§4, 06:§1]
- **Resolution**: свежая и объёмная campaign data авторитетнее. Fit есть, но НЕ на geo-corridor level — на **business-model level**: UAE HQ + agency-service + команда разбросана по Ливану/ЕС/IN/PK одновременно. UAE→India одиночным корридором — нет fit. UAE HQ с 5-country payout pattern — есть fit. [01:§7.1]

### 9.2 Target = C-level vs Finance Ops ✅ resolved
- **Source A** (02 EN site): CTA "Book a demo", tone к CEO/CFO/founder
- **Source B** (04 named buyers + 01 warm replies): Finance Ops / Accountant / BizDev
- **Source C** (06 DACH Ilya V.): C-level работает в DACH (6.82% reply)
- **Resolution**: **Segment-dependent**. Gulf/CIS/iGaming → Finance Ops/Accountant. DACH exception → CEO/CFO. Default filter должен быть Finance Ops для >85% pipeline, с DACH override.

### 9.3 Pricing "<1%" vs "3%/$39" ⚠️ unresolved (product-team level)
- **Source A** (Step 1 template): "fees <1%" [01:§7.5]
- **Source B** (Step 3 + pricing page): "from 3% or $39/task" [02:§5]
- **Resolution**: "<1%" — high-volume tier (>€100K/mo turnover); "3%/$39" — стартовая публичная цена. **Не править без согласования команды** (Step 3 даёт 14% warm). Возможный fix в копии: "starting from 3% / $39 per task — <1% at volume".

### 9.4 Sender "Petr" vs Eleonora vs Ilya V. ✅ resolved
- Petr/Rinat (launched Mar 22) — НЕ warm history, low reply rate 0.3–1% [06:§4]
- Eleonora/Marina — historical warm performer 2–4%
- Ilya V. DACH — peak performer 6.82% но только в DACH C-level contexте
- **Resolution**: Petr/Rinat нельзя запускать на исторически успешных corridors без A/B. Eleonora — default для Gulf/UAE/AU. Ilya V. — только DACH.

### 9.5 Apollo credits vs Puppeteer ✅ resolved
- Apollo API credits = 0 [01:§7.3]
- Apollo UI-scraper через Puppeteer = free [01:§7.3]
- **Resolution**: production = Puppeteer emulator. Нет противоречия.
