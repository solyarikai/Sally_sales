ОПТИМИЗАЦИЯ ФИЛЬТРОВ ДЛЯ INFLUENCER MARKETING\n\n### 3.1 Анализ текущих keywords OnSocial\n\n**Текущие keywords достаточны на ~70%.** Пропущенные категории:\n\n**Критически пропущенные (высокая релевантность):**\n- `social commerce platform` -- Bazaarvoice, LTK, ShopMy\n- `creator economy` -- общий тренд-термин, широко используется\n- `creator fund` / `creator monetization` -- платформы монетизации\n- `UGC platform` / `user generated content platform` -- Insense, Trend, Social Native\n- `brand ambassador platform` -- BrandChamp, CrewFire\n- `creator management` / `talent management platform`\n- `social media management` -- частично пересекается, но есть overlap (Sprout Social, Hootsuite продают influencer tools)\n\n**Важные для Segment 2 (Affiliate/Performance):**\n- `partner marketing platform` -- PartnerStack, impact.com\n- `partnership management` -- активный термин в индустрии\n- `creator commerce` / `social selling`\n- `referral marketing platform`\n- `performance partnership`\n\n**Важные для Segment 3 (Agencies):**\n- `TikTok marketing agency` (конкретнее чем \"TikTok agency\")\n- `creator management agency`\n- `social media agency` -- с дополнительной фильтрацией\n- `content creator agency`\n- `brand partnership agency`\n\n**Emerging terms (2025-2026):**\n- `AI influencer marketing` -- AI-powered campaign optimization\n- `virtual influencer` / `CGI influencer` -- новая ниша\n- `live shopping` / `livestream commerce`\n- `short-form video marketing`\n- `creator analytics` -- уже есть, но убедиться что покрывает \"creator intelligence\"\n\n### 3.2 Description-based search vs Keyword search\n\n**Рекомендация: использовать ОБА параллельно.**\n\nПричина: Apollo Keywords -- это теги, сгенерированные AI. Если AI не распознал компанию, тег не назначен. Description search ищет по SEO-описанию и bio в соцсетях, которые компания пишет сама.\n\nСтратегия:\n1. **Run 1:** Keyword search (текущий подход) -- даёт основной объём\n2. **Run 2:** Description search по тем же словам -- находит компании, которые не были протегированы, но описывают себя как \"influencer marketing\"\n3. **Дедупликация** через pipeline автоматически уберёт пересечения\n\n### 3.3 Проблема индустриальных тегов\n\nМногие influencer marketing компании имеют **неправильный или неполный индустриальный тег**:\n\n| Реальный бизнес | Частые теги в Apollo |\n|---|---|\n| CreatorIQ (платформа) | \"Computer Software\", \"Internet\" |\n| Viral Nation (агентство) | \"Marketing and Advertising\" |\n| impact.com (affiliate) | \"Computer Software\", \"Internet\" |\n| The Shelf (платформа) | \"Marketing and Advertising\", \"Internet\" |\n| Grin (платформа) | \"Computer Software\" |\n\n**Рекомендация:** НЕ полагаться только на индустрию. Текущий подход OnSocial (индустрия + keywords) правильный, но нужно добавить отдельный run только по keywords БЕЗ индустриального фильтра -- это поймает компании с неправильным тегом индустрии.\n\n---\n\n## 4.
