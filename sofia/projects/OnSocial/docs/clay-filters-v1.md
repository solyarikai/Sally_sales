## Сегмент 1: INFLUENCER_PLATFORMS

### ICP Text (для Clay AI Search)

```
SaaS-компании, которые строят продукты в области influencer marketing и creator economy.
Это платформы для поиска инфлюенсеров, аналитики аудитории, creator marketplace,
social listening, UGC-платформы, инструменты для работы с creator data API.

Примеры компаний: Modash, Captiv8, Lefty, Kolsquare, Skeepers, Favikon, Phyllo,
KlugKlug, The Shelf, impact.com.

НЕ интересуют: рекрутинговые агентства, PR-агентства, generic SMM-агентства,
разработка сайтов, SEO/PPC-агентства, консалтинг, финтех, стартапы без продукта.

Ключевая характеристика: у компании ЕСТЬ собственный технологический продукт
(платформа, SaaS, API) в области influencer/creator данных или аналитики.
```

## Заметки для оператора

1. **Clay vs Apollo:** Clay использует семантический AI-поиск (ICP Text), а не просто keyword-теги. Поэтому ICP Text — главный фильтр, keywords — дополнительный.
2. **Ocean.io Lookalike:** Загрузить 10 seed-доменов в Clay → Enrichment → "Find Similar Companies". Даёт +5-10% нишевых компаний, пропущенных keyword-поиском.
3. **Приоритет запуска:** Сначала IM_FIRST_AGENCIES (наибольший TAM, наибольший рост). Затем INFLUENCER_PLATFORMS (TAM частично исчерпан, но MENA/India — новые).
4. **AFFILIATE_PERFORMANCE не включён** — конфигурация для Clay будет подготовлена отдельно, когда подтвердится эффективность первых двух сегментов.
5. **Все запуски через gathering pipeline** с 3 чекпоинтами. Никаких ad-hoc скриптов.
