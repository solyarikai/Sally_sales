## Продукт / Услуги

AI-powered платформа для работы с creator-данными. Не campaign management tool — позиционируют себя как data layer для тех, кто уже строит свои продукты.

1. **Creator Discovery** — все публичные аккаунты 1000+ подписчиков на IG, TikTok, YouTube
2. **Audience Analytics** — демография, интересы, overlap аудиторий
3. **Sponsored Posts Tracking** — история коллабораций
4. **Raw API Access** — интеграция в сторонние продукты
5. **White-label** — ребрендинг для агентств и платформ

**Ценообразование:** custom, pay-per-successful-request, без публичных тарифов — demo → sales call

## Выводы для аутрича

**Главный хук:**

> "You're building an influencer product — but spending months scraping creator data yourself."

**На что делать акцент в письмах:**

- API-first: OnSocial = инфраструктура, не конкурент чужим продуктам
- White-label — агентства перепродают под своим брендом без затрат на разработку
- Pay-per-request — низкий порог входа, нет контракта на минимум

**Приоритетный сегмент:** SaaS-платформы и агентства — им нужна data-инфраструктура, не ещё один инструмент
