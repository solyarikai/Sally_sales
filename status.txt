## API Endpoints

### Reply Automation
- GET /api/replies/ - List replies with filters
- GET /api/replies/stats - Get reply statistics
- GET /api/replies/{id} - Get specific reply
- POST /api/replies/{id}/resend-notification - Resend Slack notification
- GET /api/replies/automations - List automations
- POST /api/replies/automations - Create automation
- PATCH /api/replies/automations/{id} - Update automation
- DELETE /api/replies/automations/{id} - Delete automation

### Smartlead
- GET /api/smartlead/campaigns - List campaigns (READ-ONLY)
- GET /api/smartlead/campaigns/{id} - Get campaign (READ-ONLY)
- GET /api/smartlead/campaigns/{id}/leads - Get leads (READ-ONLY)
- POST /api/smartlead/webhook - Receive reply events
- POST /api/smartlead/simulate-reply - Test reply processing

### Slack
- POST /api/slack/interactions - Handle button clicks

## Configuration

### Required Environment Variables
- SLACK_BOT_TOKEN - Slack Bot OAuth token
- OPENAI_API_KEY - OpenAI API key
- SMARTLEAD_API_KEY - Smartlead API key

### Smartlead Webhook Setup
Configure webhook URL in Smartlead:
`https://your-domain.com/api/smartlead/webhook`
