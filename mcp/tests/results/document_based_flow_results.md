## Final Architecture

### Three Dynamic Agents (no hardcoding)

1. **Document Extractor** (gpt-4.1-nano): reads ANY document → structured JSON
2. **Classification Prompt Generator** (gpt-4.1-mini): creates optimal prompt for pipeline
3. **Streaming Pipeline Classifier** (gpt-4o-mini): classifies each company using generated prompt
